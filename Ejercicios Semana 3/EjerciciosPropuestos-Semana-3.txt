1- Número de Kaprekar
	- Pedir al usuario un número de 4 dígitos con alguna cifra distinta N
	- Se ordena de forma creciente C y decreciente D
	- Se calcula D - C
	- Con ese número se hacen las mismas operaciones
	- En algún momento, N = D - C, con lo que ya siempre se repetirá el número
	- NOTA: Si un número tuviese MENOS de 4 dígitos, se complementaría con CEROS por la izquierda
	- El sistema devuelve 
		- La operación que se hace está haciendo un listBox
		- el número de iteraciones necesarias para conseguirlo
		- El valor de N

	- Ejemplo: El número que elige el usuario es N=3524:
	    5432 – 2345 = 3087
	    8730 – 0378 = 8352
	    8532 – 2358 = 6174
	    7641 – 1467 = 6174 
	Se ha hallado en 3 iteraciones y el número es 6174
